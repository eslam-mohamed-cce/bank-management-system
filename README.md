# bank-management-system

A console-based Bank Management System developed using C programming language.  
The system simulates basic banking operations and user management.

Features:
- Create new bank account
- Add new users
- Delete accounts
- Deposit money
- Withdraw money
- View account details
- Manage customer data

Technologies Used:
- C Programming Language
- File handling 
- Basic data structures (arrays / structs)

Project Purpose:
This project was developed as part of Programming 1 course to practice problem solving, file handling, and basic system design concepts.

How to Run:
Compile the project using any C compiler (e.g., GCC or CodeBlocks) and run the main file.
