# bank-management-system-c


A console-based Bank Management System developed using the C programming language.  
This project simulates real banking operations and manages customer accounts.

Features:
- Add new account
- Delete account
- Modify account details
- Search for account
- Advanced search (by multiple criteria)
- Deposit money
- Withdraw money
- Transfer money between accounts
- Change account status (active/inactive)
- Print all accounts
- Generate simple reports
- Save data and quit system

Technologies Used:
- C Programming Language
- File Handling
- Structures and Functions

Project Purpose:
This project was developed as part of Programming 1 course to practice problem solving, data structures, file handling, and system design in C.

How to Run:
Compile the project using GCC or CodeBlocks, then run the main file (main.c).
